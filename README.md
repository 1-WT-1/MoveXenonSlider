# Move Xenon Slider

Moves the Xenon usage slider from the Tuning screen to the Dive Selection screen.

> [!WARNING]
> HevLib is required for this mod to function.
> A download can be found from its [releases page](https://github.com/rwqfsfasxc100/HevLib/releases/latest)

## Features

* Moves the Xenon usage slider to the Dive Selection screen, allowing you to tweak Xenon usage on the fly when choosing your dive target.
